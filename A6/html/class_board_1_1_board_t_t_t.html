Detailed Description

Specific board class for TTT game board class for TTT which is a subclass of board, this subclass contains methods specific to TTT, such as detecting the win scenario by working out adjacent squares to the players current square.

int Board.BoardTTT.detectEndGame ( )

First checks for a draw game by looping though every square and checking if there are any empty squares, if there is not then the game is a draw. It then calls the win() method on both X and O
